Python

在python中变量是没有类型的，但是变量存储的数据是有类型的。

想要将字符串转换成数字，必须要求字符串的内容都是数字

字符串拼接只能和字符串本身拼接，无法和非字符串类型进行拼接

.n会对小数部分做精度限制，同时会对小数部分做四舍五入

输出不换行在print里面加上一个逗号end=‘’

for循环语句，本质上遍历：序列类型

语法一：range(num)

获取一个从0开始，到num结束的数字序列（不含num本身）

如，range（5）取得的数据是：[0，1，2，3，4]

语法二：range(num1,num2)

获得一个从num1开始，到num2结束的数字序列（不含num2本身）

如，range(5，10)取得的数据是：[5，6，7，8，9]

语法三：range(num1，num2，num3)

获得一个从num1开始，到num2结束的数字序列（不含num2本身）

数字之间的步长以step为准，默认情况下为1

如，range(5，10，2)取得的数据是：[5，7，9]

continue：在循环中，因某些原因，临时结束本次循环

None

  • 用在函数无返回值上
  • 用在if判断上
    • 在if判断上，None等同于False
    • 一般用于在函数中主动返回None，配合if判断做相关处理
  • 用于声明无内容的变量上
    • 定义变量，但暂时不需要变量有具体值，可以用None来代替

global关键字，在函数内声明变量为全局变量

数据容器根据特点的不同：

  • 是否支持重复元素
  • 是否可以修改
  • 是否有序

列表、元组、字符串、集合、字典

1.1 查找某元素在列表内下标索引

index = mylist.index()

1.2 修改特定下标索引的值

mylist[0] = “”

1.3 在指定下标位置插入新元素

mylist.insert()

1.4 在列表的尾部追加单个新元素

mylist.append()

1.5 在列表尾部追加一批新元素

mylist2 = []

mylist.extend(mylist2)

1.6 删除指定下标索引的元素（2种方式）

del mylist[2] element = mylist.pop(2)

1.7 删除某元素在列表中的第一个匹配项

mylist.remove

1.8 清空列表

mylist.clear()

1.9 统计列表内某元素的数量

count = mylist.count

1.10 统计列表内全部元素数量

count = len(mylist)

元组一旦定义完成，就不可修改

应用场景：当我们需要在程序内封装数据，又不希望封装的数据被篡改，那么元组就非常适合

统计元组内的元素个数

len()

字符串是一个无法修改的数据容器

修改指定下标的字符+移除特定下标的字符+追加字符等都无法做到

  • 查找特定字符串的下标索引

    value = my_str[-16]

    负数代表从后向前找

    my_str.index()

  • 字符串的替换

    字符串的替换不是修改字符串本身，而是得到了一个新字符串

    my_str.replace()

  • 字符串的分割

    字符串本身不变，而是得到了一个列表对象

    my_str.strip()

    不传入参数，去除首尾空格

  • 字符串的规整操作

    去空格：my_str_list = my_str.split(“ “)

    去字符串：my_str_list = my_str.split(“12”)

序列：内容连续、有序，可使用下标索引的一类数据容器

列表、元组、字符串均可以视为序列

切片不会影响序列本身，而是会得到新的序列

  • 添加新元素

my_set.add(”Python“)

  • 移除元素

my_set.remove()

  • 随机取出一个元素

element = my_set.pop()

  • 清空集合

my_set.clear()

  • 取2个集合的差集

set = set1.difference(f”{set2}“)

  • 消除2个集合的差集

set = set1.difference_update(set2)

  • 2个集合合并为1个

set3 = set1.union(set2)

  • 统计集合元素数量len()

len()

集合的遍历

集合不支持下标索引，不能用while循环

可以用for循环

定义重复key的字典

从字典中基于Key获取Value

  • 新增元素

stu_score[“ “] = 66

  • 更新元素

stu_score[“ “] = 66

注意：字典key不可以重复，所以对已存在的Key执行上述操作，就是更新Value值

  • 删除元素

my_dict.pop()

  • 清空元素

my_dict.clear()

  • 获取全部的key

my_dict.keys()

  • 遍历字典

for key in keys:

​ 字典的key：{key}

​ 字典的value：{my_dict[key]}

for key in my_dict:

​ key:{key}

​ value:{my_dict[key]}

  • 统计字典内的元素数量

len()

数据容器可以从以下视角进行简单的分类：

  • 是否支持下标索引

    • 支持：列表、元组、字符串 - 序列类型
    • 不支持：集合、字典 + 非序列类型

  • 是否支持重复元素

    • 支持：列表、元组、字符串 - 序列类型
    • 不支持：集合、字典 - 非序列类型

  • 是否可以修改

    • 支持：列表、集合、字典
    • 不支持：元组、字符串

  • 列表：一批数据，可修改、可重复的存储场景

  • 元组：一批数据，不可修改、可重复的存储场景

  • 字符串：一串字符串的存储场景

  • 集合：一批数据，去重存储场景

  • 字典：一批数据，可用key检索Value的存储场景
